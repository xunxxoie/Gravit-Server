# [REQ-007] 축하하기

## 개요
친구 활동 피드에서 축하하기 버튼을 눌러 해당 유저에게 LP를 지급하고 알림을 전송한다.

## API
| Method | Path | 설명 |
|--------|------|------|
| POST | /api/v1/social/feed/{feedId}/congratulate | 피드 항목 축하하기 |

## 비즈니스 규칙
- 축하받은 유저에게 5 LP 지급 (시즌 LP에 합산)
- 축하받은 유저에게 인앱 알림 발송: "OO님이 축하해줬어요!"
- 동일 유저에게 하루 최대 3회까지 축하 가능
- 하루 3회 초과 시:
  - 해당 피드 항목의 축하하기 버튼 비활성화 (회색)
  - 클릭 시 토스트 메시지: "오늘 축하 횟수를 모두 사용했어요"

## 참고
- 관련 엔티티: Congratulation, SocialFeed, UserLeague
- 관련 도메인 패키지: social
