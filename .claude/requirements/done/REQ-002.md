# [REQ-002] 시즌 주기 변경

## 개요
기존 1주 단위 시즌을 4개월 단위로 변경하여 유저가 상위 티어에 도달할 수 있는 충분한 기간을 확보한다.

## API
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/leagues/season/current | 현재 시즌 정보 조회 |

## 비즈니스 규칙
- 시즌 주기: 4개월
- 시즌 ID 형식: `2026-S1` (연도-시즌번호, 기존 `2026-W20` 형식 변경)
- 시즌 종료 시 소프트 리셋 적용 (REQ-003 참고)
- 시즌 종료 후 첫 접속 시 결과 팝업 노출
  - 표시 내용: 직전 시즌 최종 티어, 다음 시즌 시작 티어 및 시작 LP
  - 팝업은 첫 접속 1회만 노출 (이미 확인한 유저 재노출 없음)

## 참고
- 관련 엔티티: LeagueSeason
- 관련 도메인 패키지: league
