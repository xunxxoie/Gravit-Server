# [REQ-008] 추천 친구

## 개요
소셜 탭 하단에 팔로우하지 않은 유저를 추천하여 소셜 네트워크 확장을 유도한다.

## API
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/social/recommend | 추천 친구 목록 조회 |
| POST | /api/v1/social/follow/{userId} | 팔로우 |

## 비즈니스 규칙
- 위치: 프로필 > 소셜 탭 하단 "비슷한 레벨의 학습자들"
- 추천 기준:
  - 기본: 나와 같은 메인 티어 유저
  - 폴백: 같은 메인 티어 유저가 5명 미만일 경우 ±1 메인 티어로 확장
  - 이미 팔로우한 유저 제외
  - 본인 제외
- 추천 유저 카드에 닉네임, 팔로우 버튼 노출
- 팔로우 버튼 클릭 시 즉시 팔로우 처리
- 최대 8명 노출, 노출 순서: 랜덤

## 참고
- 관련 엔티티: Follow, UserLeague
- 관련 도메인 패키지: social
