# 테스트 실행

## 실행 방법

- 인자가 없으면 전체 테스트를 실행하라: `./gradlew test`
- 인자가 있으면 해당 클래스/메서드만 실행하라: `./gradlew test --tests "$ARGUMENTS"`

## 실패 시 대응

1. 실패 로그를 분석하라
2. 실패 원인을 사용자에게 간결하게 설명하라
3. 수정 방안을 제안하되, 사용자 확인 없이 코드를 수정하지 마라
