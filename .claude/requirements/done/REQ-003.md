# [REQ-003] 소프트 리셋

## 개요
시즌 종료 시 LP를 전체 초기화하지 않고, 직전 시즌 티어를 기반으로 다음 시즌 시작 티어와 시작 LP를 차등 지급한다.

## API
없음 (시즌 종료 시 내부 스케줄러로 자동 실행)

## 비즈니스 규칙
- 메인 티어 1단계 하락, 서브 단계(1/2/3) 유지
  - 예: 골드 1 → 실버 1 / 플래티넘 2 → 골드 2
- 브론즈는 강등 없음 (브론즈 유지)
- 시작 LP는 아래 매핑 테이블을 따름

| 직전 티어 | 시작 티어 | 시작 LP |
|-----------|-----------|---------|
| 브론즈 3 | 브론즈 3 | 0 |
| 브론즈 2 | 브론즈 3 | 0 |
| 브론즈 1 | 브론즈 3 | 50 |
| 실버 3 | 브론즈 2 | 101 |
| 실버 2 | 브론즈 2 | 150 |
| 실버 1 | 브론즈 1 | 201 |
| 골드 3 | 실버 3 | 321 |
| 골드 2 | 실버 2 | 461 |
| 골드 1 | 실버 1 | 621 |
| 플래티넘 3 | 골드 3 | 801 |
| 플래티넘 2 | 골드 2 | 1001 |
| 플래티넘 1 | 골드 1 | 1221 |
| 다이아 3 | 플래티넘 3 | 1461 |
| 다이아 2 | 플래티넘 2 | 1721 |
| 다이아 1 | 플래티넘 1 | 2001 |

- 별도 보상 시스템 없음 (시작 위치 자체가 보상)

## 참고
- 관련 엔티티: UserLeague, LeagueSeason
- 관련 도메인 패키지: league
