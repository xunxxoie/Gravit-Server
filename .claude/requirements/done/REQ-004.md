# [REQ-004] 리그 구조

## 개요
전체 유저가 하나의 리그에서 LP 기반으로 순위 경쟁한다. 티어 변동은 시즌 종료 시 소프트 리셋으로만 처리한다.

## API
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/leagues/ranking | 전체 리그 순위 조회 |

## 비즈니스 규칙
- 전체 유저가 하나의 리그에서 경쟁 (별도 그룹 분리 없음)
- 승강급 없음 — 티어 변동은 시즌 종료 시 소프트 리셋으로만 적용 (REQ-003 참고)
- 티어 구조: 브론즈 3 → 브론즈 2 → 브론즈 1 → 실버 3 → 실버 2 → 실버 1 → 골드 3 → 골드 2 → 골드 1 → 플래티넘 3 → 플래티넘 2 → 플래티넘 1 → 다이아 3 → 다이아 2 → 다이아 1

## 참고
- 관련 엔티티: UserLeague, LeagueSeason
- 관련 도메인 패키지: league
