# [REQ-005] League History

## 개요
유저의 시즌별 최종 티어 기록을 프로필 내 리그 탭에서 확인할 수 있도록 노출한다.

## API
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/leagues/history | 내 리그 히스토리 조회 |
| GET | /api/v1/leagues/history/{userId} | 특정 유저 리그 히스토리 조회 |

## 비즈니스 규칙
- 위치: 프로필 > 리그 탭
- 표시 항목:
  - 현재 시즌 순위
  - 총 참여 시즌 수
  - 역대 최고 티어
  - 시즌별 티어 변화 (S1 ~ 현재)
- 첫 시즌 참여 유저는 현재 시즌 데이터만 표시

## 참고
- 관련 엔티티: LeagueHistory, UserLeague, LeagueSeason
- 관련 도메인 패키지: league
